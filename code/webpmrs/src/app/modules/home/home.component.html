Project Management System

Project Management and Repository System is a web-application that helps Admins, project Mentors and project group members to collaborate and meet goals on time while managing resources and evaluating final statics.

Admin can add new department, mentors and students. He also approves projects groups, assigns mentors to project and track project status.

Project Mentor view the list of projects under him. He also track project status, verifies docs uploaded by prject group at various stages during project life cycle.

Students create project groups. Once the project group is apporved by the admin, they can upload documents like SRS etc. at various stages of project.

Project management and Repository System is also referred to as Task Management Software.
